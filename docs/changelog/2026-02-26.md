## Figma HTML-to-Design キャプチャスクリプト追加

### 変更ファイル

| ファイル | 変更内容 |
|---------|---------|

### 詳細

`<head>` セクションに Figma の HTML-to-Design キャプチャスクリプト（`capture.js`）を `async` で読み込むよう追加。
Figma MCP サーバーとの連携により、開発中の画面をFigma側にキャプチャ・同期するためのもの。

```html
<head>
  <script src="https://mcp.figma.com/mcp/html-to-design/capture.js" async></script>
</head>
```

- Next.js の `@next/next/no-sync-scripts` lint ルールを `eslint-disable` で明示的に無効化
- 開発・デザインワークフロー用途（本番影響なし）
