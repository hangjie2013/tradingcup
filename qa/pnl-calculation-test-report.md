## テスト対象の仕様根拠（CLAUDE.md セクション 4）

```
開始総資産 = USDT残高 + IZKY残高 × 開始時点価格    (4.1)
終了総資産 = USDT残高 + IZKY残高 × 終了時点価格    (4.2)
PNL = 終了総資産 − 開始総資産                       (4.4)
PNL% = (PNL ÷ 開始総資産) × 100                    (4.5)
```

### シナリオ C（未決済ポジション）の詳細

CLAUDE.md 4.3 に基づく最重要テストケース:

```
取引:
  BUY  1,000,000 IZKY @ 0.0010 = -1,000 USDT
  SELL   200,000 IZKY @ 0.0015 = +300 USDT

決済ベース利益: +100 USDT（売却益のみ）
残 IZKY: 800,000 @ 終了時点価格 0.0008 = 640 USDT
未決済含み損: 800,000 × (0.0008 - 0.0010) = -160 USDT

終了総資産: 300 + 640 = 940 USDT
PNL: 940 - 1,000 = -60 USDT
PNL%: -6.00%
```

決済利益だけ見ると +100 USDT だが、未決済ポジションの含み損を反映すると **-60 USDT** が正しい。
